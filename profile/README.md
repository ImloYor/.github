# ImloYor

**Open-source spellcheck tools for the Tajik language.**

ImloYor provides a comprehensive Hunspell dictionary with 247,000+ word forms for Tajik (Cyrillic script), along with integrations for browsers, office suites, and a web-based spell checker.

## Projects

| Repository | Description |
|---|---|
| [dictionary](https://github.com/ImloYor/dictionary) | Hunspell dictionary files (tg_TJ.dic + tg_TJ.aff), frequency data, and bigram model |
| [firefox-dictionary](https://github.com/ImloYor/firefox-dictionary) | Firefox spellcheck dictionary extension ([Firefox Add-ons](https://addons.mozilla.org/firefox/addon/imloyor-tajik-dictionary/)) |
| [libreoffice-extension](https://github.com/ImloYor/libreoffice-extension) | LibreOffice / OpenOffice dictionary extension (.oxt) |

## Data Sources

- **uniparser-grammar-tajik** (MIT) — morphological paradigms and corpus frequencies
- **kumitaizabon.tj** — Official orthographic dictionary from the Committee on Language and Terminology of Tajikistan
- **Chromium Hunspell dictionaries** — 34K additional words from Chrome's tg_TG dictionary
