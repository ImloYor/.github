# ImloYor

**Open-source spellcheck tools for the Tajik language.**

ImloYor provides a comprehensive Hunspell dictionary with 213,000+ word forms for Tajik (Cyrillic script), along with integrations for browsers, office suites, and a web-based spell checker.

## Projects

| Repository | Description |
|---|---|
| [dictionary](https://github.com/ImloYor/dictionary) | Hunspell dictionary files (tg_TJ.dic + tg_TJ.aff) and generation pipeline |
| [firefox-dictionary](https://github.com/ImloYor/firefox-dictionary) | Firefox spellcheck dictionary extension |
| [chrome-extension](https://github.com/ImloYor/chrome-extension) | Chrome spellcheck extension |
| [libreoffice-extension](https://github.com/ImloYor/libreoffice-extension) | LibreOffice / OpenOffice dictionary extension (.oxt) |


## Data Sources

- **uniparser-grammar-tajik** (MIT) — 211K lexemes with morphological paradigms
- **kumitaizabon.tj** — Official orthographic dictionary from the Committee on Language and Terminology of Tajikistan
